# 022 — 本地化变现策略

概述：根据不同市场（中国/东南亚/欧美）制定差异化变现策略，兼顾合规与文化适配。

关键功能点：
- 市场分层策略：国内以道具/皮肤/订阅为主，国际市场兼顾广告与一次性付费
- 活动付费节奏：节日限定礼包、主题皮肤、季票（Battle Pass）与限时促销
- 支付渠道与合规：国内接入微信/支付宝，国际接入 Google Play/Apple；税务与发票考量
- 价格区分策略：基于地域/机型/ARPU 做差异化定价与本地化文案
- 道具设计原则：不破坏平衡、非付费玩家仍能通过时间/策略获得核心体验

优先级：P2（上线前确定）

验收标准：
- 明确各目标市场的变现产品清单与接入需求
- 与支付/法务确认接入与合规流程

实现要点（PM 视角）：
- 设计变现前后对体验影响的监测指标与回退策略
- 制定促销日历并与内容更新/节日活动联动

备注：定价与文案本地化直接影响付费转化率，需与市场团队紧密合作。
