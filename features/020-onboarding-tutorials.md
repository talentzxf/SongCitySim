# 020 — 新手引导与教学体系

概述：构建分层的新手教学流程，帮助玩家在不同阶段掌握核心玩法并激发长期目标。

关键功能点：
- 分段式教学任务：初始建城、基本税收、道路与物流、节日活动等逐步教学
- 可跳过/可回放教程：为经验玩家提供快速通道，亦允许回放学习特定系统
- 内嵌帮助面板：上下文相关的帮助（工具提示、系统观看示例）
- 引导策略：新手保护期（降低灾害/成本）、引导奖励（免费建筑/资源）
- 教学数据埋点：教程完成率、卡关点、掉线/流失点

优先级：P0

验收标准：
- 新玩家在 30 分钟内完成核心教学并达到可自我管理的基础城市状态
- 教程埋点能形成数据报表（完成率/放弃步骤）

实现要点（PM 视角）：
- 教学内容与任务脚本需要可配置并可在后期更新
- 设计低挫败感的教学目标与即时回馈机制

备注：良好教学是高留存的关键，优先持续迭代。
