# 032 — 存档格式与兼容性规范

概述：定义存档（savegame）格式、版本迁移策略与增量同步协议，保障跨平台一致性与向后兼容。

关键功能点：
- 存档格式规范：JSON 可读/二进制压缩两套方案，字段约束与资源引用方式
- 版本兼容策略：每次变更须写迁移脚本，旧版本存档自动迁移或告知回退
- 增量与云同步：支持差分上传、冲突检测与合并策略（最后写入/用户选择）
- 本地备份与回滚：自动保存历史快照，允许用户回滚到指定版本
- 隐私与加密：敏感字段加密，跨设备传输采用安全通道

优先级：P0

验收标准：
- 提供存档格式文档并实现旧版迁移的示例脚本
- 云同步能处理一个并发冲突场景并提供用户冲突解决界面

实现要点（PM）：
- 定义存档容量/大小上限并纳入性能测试
- 明确跨平台资源 ID 的映射规则，防止资源差异导致错配

备注：存档兼容性直接影响用户信任与长期运维复杂度。
