# 003 — 文化与中国风建筑

概述：把中国风（宋/明/清/民国/现代）文化元素作为玩法驱动：节庆、建筑风格、民俗事件、名人/历史传说、戏曲与市井生活。

关键功能点：
- 多时代/多地域建筑皮肤与城市主题（江南水乡、北方古城、岭南侨乡）
- 文化指标：文化值、传统认同、旅游吸引力
- 文化事件/活动：庙会、元宵灯会、端午龙舟、清明踏青等，带来游客与经济增益
- 文化遗产与名胜：可以规划保护区，影响旅游/科研
- 手工艺与匠人系统：作坊产出特色手工品，作为贸易/旅游商品

优先级：P1

验收标准：
- 至少实现一种节庆事件并能看到经济/幸福/游客变化
- 建筑主题皮肤能切换并在视图中体现

实现要点：
- 文化活动用事件系统驱动（触发条件、持续时间、奖励/惩罚）
- 视觉与音效资源需配合（鼓乐、笛子、锣声等）

依赖：002 建筑系统，001 模拟系统
