# 034 — 第三方扩展与开发者 SDK

概述：为第三方开发者提供 SDK 与 API，允许构建工具、统计插件或其他扩展，构建生态。

关键功能点：
- 只读公开 API：读取城市快照/排行榜/公共市场数据的 RESTful 接口
- 本地开发 SDK：Node/TypeScript SDK 提供模组操作、事件导入导出工具
- Webhooks 与事件订阅：支持外部系统接收特定事件（城市上榜/模组发布/付费）
- 权限与配额：对第三方调用进行速率限制与权限控制
- 文档与示例项目：示范如何使用 SDK 构建工具/分析仪表盘

优先级：P3（长期生态）

验收标准：
- 发布基础 SDK（TS）与 2 个示例项目（快照展示/数据分析）
- 提供 API Key 管理与使用限额文档

实现要点（PM）：
- 先以只读 API 起步，确保安全后逐步开放写入能力
- 制定第三方接入条款与审核流程

备注：生态是长期战略，早期可先做沙箱与只读能力验证。
