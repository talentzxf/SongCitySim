# 019 — Mod 支持与关卡编辑器

概述：为玩家/社区提供编辑器与 Mod 支持，延长生命周期与社区创作活跃度。

关键功能点：
- 基础关卡/场景编辑器：地形/建筑/事件/NPC 布置、可视化调参
- 事件脚本导入/导出：支持 JSON/YAML 格式的事件与任务脚本
- 模组托管与分享：内置模组市场或社区仓库（只读预览/作者信息/评分）
- 安全沙箱：运行模组时限制资源访问与性能消耗、防止恶意脚本
- 官方认证模组机制：可标注“官方认证”与付费分发选项

优先级：P3（长期）

验收标准：
- 发布基础编辑器并允许导出/导入一个自定义场景
- 社区能上传并分享第一个模组（审查/签名流程）

实现要点（PM 视角）：
- 明确编辑器的目标用户（创作者/普通玩家）与最小功能集
- 制定模组审核与分发政策

备注：Mod 社区增长对长期留存与营收有显著帮助。
