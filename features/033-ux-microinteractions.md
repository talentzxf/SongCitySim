# 033 — 细节交互（Microinteractions）规范

概述：制定细节交互规范（动效、反馈、过渡），提升界面亲和力与操作可理解性。

关键功能点：
- 动效规范：按钮按压、建筑建造完成、资源收集的视觉/听觉反馈
- 过渡与缓动曲线：统一缓动曲线库（ease-in/out 等）与时长标准
- 错误与确认流程：操作前置确认、撤销/重做支持的时限与提示
- 微观提示：悬停/长按提示、信息层级与优先级规则
- 可配置性：允许低性能模式关闭非关键动效

优先级：P1

验收标准：
- 发布微交互库并在主要 UI 流程中实现示例
- 低性能模式下关键功能仍可正常操作且动效可关闭

实现要点（PM）：
- 与设计/前端团队约定动效验收清单
- 将微交互纳入可用性测试指标

备注：优秀的微交互能显著提高产品“质感”。
