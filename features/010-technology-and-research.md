# 010 — 科技树与研发

概述：通过研发解锁新建筑/政策/设备，驱动城市长期策略与特色发展（如水利工程、印刷术、航海技术）。

关键功能点：
- 科技树：基础（农业/城防/基础设施）→中级（商业/文化）→高级（工业化/近代化）
- 研究投入：预算分配、院校/学舍加速研究
- 发明/技术事件：推荐发明给城市长期加成
- 技术溢出：与其他城市/商队交换技术或盗取技术（事件）

优先级：P2

验收标准：
- 玩家能投资研究并解锁至少 3 个科技节点
- 科技解锁后建筑或政策在游戏中体现效果

实现要点：
- 科技节点以数据驱动，具备描述、前置条件、成本、效果
- 可扩展为多人竞速/竞赛模式
