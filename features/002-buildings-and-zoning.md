# 002 — 建筑与分区

概述：支持中国风建筑种类、分区系统（居住、商业、工业、文化）、建筑升级与外观自定义。

关键功能点：
- 建筑分类：小商铺、行宫、四合院、民居、茶楼、寺庙、牌坊、作坊、粮仓、学舍等
- 分区规则：指定地块用途与税收、拥堵影响
- 建筑升级与连片效果（如一整片四合院影响幸福度+文化值）
- 装饰与景观：园林、石桥、假山、荷塘、牌楼、红灯笼等可装饰物
- 建筑美术风格替换（朝代/地域皮肤）
- 可自定义建筑名、外观细节（如窗格、瓦当颜色）

优先级：P0（基础建筑）、P1（皮肤/深度装饰）

验收标准：
- 玩家能建造不同用途建筑并看到对应效果（人口、税收、幸福）
- 装饰能影响美观/文化相关指标

实现要点：
- 模块化建筑元数据（cost、populationImpact、serviceSupply、visualAssets）
- 使用图块/瓦片系统或自由摆放（后期支持路径感应）

依赖：001 核心模拟系统
