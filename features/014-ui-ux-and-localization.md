# 014 — 界面、可用性与本地化

概述：面向玩家友好的 UI/UX，支持中文（简/繁）、英文等多语言，适配触摸屏与大屏（桌面）的交互。

关键功能点：
- 响应式 UI：桌面与移动自适应布局
- 引导与教学：可选的新手引导、提示系统和工具提示
- 本地化框架：文本资源、日期/数字/货币格式化、文化差异处理
- 可访问性：颜色盲模式、放大字体、键盘导航/屏幕阅读器兼容
- 快照/分享：保存城市快照并生成分享图，可分享到社交平台

优先级：P0

验收标准：
- 界面在桌面与主流手机分辨率上都能正常操作
- 完成至少中文简体与英文的文本覆盖

实现要点：
- 使用 i18n 库（例如 i18next 或自研方案）和占位符翻译流程
- 设计移动优先的交互控件（长按/拖拽/多指手势）

依赖：001、013
