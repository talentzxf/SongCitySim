# 029 — 语音与对话系统

概述：为关键 NPC、剧情与活动引入语音与对话，提升沉浸感与情绪传达。

关键功能点：
- 对话脚本规范：分条目、情绪标签、替代表达、字幕对照
- 语音实现方式：录音（配音演员）与 TTS（合成）并行方案
- 音频播放策略：优先级/并发管理、距离衰减、事件触发播放
- 多语言配音计划：优先中文并评估国际化配音策略
- 存档与回放：保存剧情语音播放记录并支持回放/剪辑生成分享短片

优先级：P2

验收标准：
- 完成 5 个关键 NPC 的语音样本和对话脚本并集成测试
- 对话系统兼容字幕和 TTS 回退

实现要点（PM）：
- 与音频导演制定配音表与录音预算
- 优先实现剧情重点段落的配音以提升体验

备注：语音成本高，推荐分阶段、小范围验证后扩展。
