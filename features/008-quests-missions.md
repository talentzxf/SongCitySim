# 008 — 剧情、任务与成就

概述：提供短期与长期任务引导玩家学习系统并创造目标感，包含历史剧本/关卡式挑战和随机委托任务。

关键功能点：
- 新手教学任务链（建城必做事项）
- 剧情关卡：历史场景还原（比如治理水患、建城扩张）
- 随机委托：商人委托、匠人任务、民间需求（修路、治水）
- 成就系统与荣誉称号（如“江南名城”）
- 可选分支与道德选择（保护古迹 vs 扩建商业）

优先级：P0（新手引导）/P1（剧情扩展）

验收标准：
- 完成基础的新手任务链并通过任务掌握核心玩法
- 随机委托能定期出现并完成奖励

实现要点：
- 任务模板（条件、目标、奖励、失败条件）
- 可扩展的剧本编辑器供后期内容创作使用
