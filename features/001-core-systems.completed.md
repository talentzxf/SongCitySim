# 001 — 核心模拟系统

概述：构建城市模拟的基本时钟与子系统，包括人口增长、需求（居住/就业/娱乐）、资源（电力/水/食物）、财政、幸福度、时间与季节流转。

关键功能点：
- 模拟循环（tick）与速度控制（暂停/慢速/正常/快进）
- 人口模型：出生/迁入/迁出/死亡、人口结构（年龄段/职业/技能）
- 需求模型：住房、工作、娱乐、教育、医疗需求表达与追踪
- 资源与基础服务：电力/水/垃圾处理/食物生产/物流消耗
- 城市等级与城市扩张：土地解锁与城市等级经验值
- 指标面板：财政、人口、失业率、幸福度、污染、犯罪等

优先级：P0（必须）

验收标准：
- 游戏可运行最小模拟循环，玩家可建造房屋/道路产生人口与税收
- 指标面板能实时反映模拟结果

实现要点：
- 后端/逻辑：TypeScript 或 C#（取决引擎），使用 ECS 风格或模块化系统
- 前端渲染与交互分离，模拟可在服务器/本地 worker 中运行

依赖：无（基础起点）

扩展想法：未来引入更复杂的经济模型、细分职业与产业链模拟。
