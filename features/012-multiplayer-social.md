# 012 — 社交与多人玩法

概述：引入社交层（好友城市、参观、排行榜）或合作/竞争多人模式（贸易联盟、争霸赛）。

关键功能点：
- 城市参观与观光：玩家互访、参观名胜、学习建筑布局模板
- 交易市场：玩家间的拍卖/交易、资源互助
- 排行与赛季：按城市评分/旅游收入/文化值等做排行榜与赛季奖励
- 联盟系统：组队共享税收/援助、共同抵抗灾害或竞争任务
- 异步多人：不强制实时交互，支持回合式或事件驱动交互

优先级：P3（可选长期目标）

验收标准：
- 实现好友互访并能看到对方城市的基本数据和快照
- 至少一个简单的异步交易或排行榜功能上线

实现要点：
- 后端需支持用户账号、存档同步、权限管理
- 考虑隐私与读写冲突，使用只读快照参观模式

依赖：001（存档）、014（账号/UI）
