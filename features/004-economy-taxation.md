# 004 — 经济与税收系统

概述：设计税收、支出、预算与财政策略，支持市政项目投资、债务与贷款、补贴与发展政策。

关键功能点：
- 税种：居住税、商业税、工业税、旅游税、交易税等
- 预算面板：收入/支出分类、财政预警、赤字与借贷
- 政策工具：税率调整、补贴、土地拍卖、招商引资
- 市政投资与项目（修桥、建学堂、修庙、护城河）带来长期收益或成本
- 黑市/逃税与腐败事件（随机或策略触发）

优先级：P0（核心）

验收标准：
- 玩家能查看并调整税率，财政数字随之变化
- 触发一次市政投资并看到长期收益/成本影响

实现要点：
- 财务模块与时间序列（每 tick 结算税收/支出）
- 可扩展的政策脚本系统来定义新政策

依赖：001 核心系统，002 建筑与人口
