# 005 — 基础设施与交通

概述：设计道路、桥梁、河道、运河、码头、马车、船只、驿站、客栈，交通将影响城市流动性与经济活动。

关键功能点：
- 路网系统：道路等级（巷道、街道、官道）、拥堵模型、维修成本
- 交通工具：人力车、马车、货运船只、渡船、驿站物流节点
- 公共交通（后期）：驿站线路、渡轮、驿使系统
- 物流与生产链：货物从作坊→仓库→市场的路径与成本
- 交通事件：堵塞、道路修缮、桥梁坍塌、漕运减速

优先级：P1

验收标准：
- 基本路网能影响市民出行时间与作坊供给
- 物流能把商品运输到市场并影响价格/库存

实现要点：
- 使用简化流量模型或 agent-based 路径查找
- 可视化拥堵热力图用于玩家决策
