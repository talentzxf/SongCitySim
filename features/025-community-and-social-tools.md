# 025 — 社区与社交工具

概述：构建社区机制，支持玩家交流、内容分享与 UGC 推广，强化社群沉淀。

关键功能点：
- 城市快照分享：生成图片/短视频并带上城市评分/标签
- 内置活动/赛事系统：周期性城市设计赛、主题挑战、排行榜
- 评论与点赞机制：对公开城市/模组的轻量互动（去中心化评分/评论）
- 官方公告与活动推送系统：在客户端内推送活动消息与奖励
- 社交账号绑定/好友系统：好友互访、互赠道具/资源

优先级：P2

验收标准：
- 实现城市快照分享功能并能在社区内查看
- 支持至少一种社区活动（设计赛）并能统计参与数据

实现要点（PM 视角）：
- 平衡社区激励与防刷机制
- 制定社区内容审核与举报流程

备注：社区是长期活跃与裂变的关键，要有早期增长计划。
